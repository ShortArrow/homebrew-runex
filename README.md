# homebrew-runex

Homebrew tap for [runex](https://github.com/ShortArrow/runex) — a cross-shell abbreviation engine that expands short tokens into full commands.

## Install

```bash
brew tap shortarrow/runex
brew install runex
```

Or in one line:

```bash
brew install shortarrow/runex/runex
```

## Supported platforms

Pre-built binaries for:

- macOS (Apple Silicon / Intel)
- Linux (x86_64 / aarch64)
